# Análise de correlação e regressão linear

3. Qual dos datasets não é apropriado para regressão linear?

Identificamos que o segundo dataset não seria apropriado para regressão linear, visto que o mesmo forma uma concavidade e, assim, uma abordagem polinomial faria mais sentido. Ao mesmo tempo, desconsideramos o terceiro dataset, já que, no caso dele, por sua natureza de dados "ruins", nenhuma abordagem faria sentido.
